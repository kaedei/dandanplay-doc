# 总结

  • VLC 内核是最省心的选择
  • EVR 内核可以加载系统中的 DirectShow 滤镜，功能更强。但需要安装解码器包才可正常使用。
  • FFmpeg 内核是 VLC 内核的最佳备选
  • 低配 PC、Windows 平板或 ARM 设备，推荐选择 WMP 或 FFmpeg 内核
  • 不建议使用 VMR9 内核
